# Alice Node

Desktop GUI for the Alice Protocol decentralized AI training network.

Built with [Tauri](https://tauri.app/) + React + TypeScript.

## Overview

Alice Node provides a graphical interface for participating in the Alice Protocol network as a miner, scorer, or aggregator.

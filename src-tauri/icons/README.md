# Icons

Generate icons using the Tauri CLI:

```bash
# From project root
npm run tauri icon public/alice-logo.svg
```

This will generate all required icon sizes in this directory:
- 32x32.png
- 128x128.png
- 128x128@2x.png
- icon.icns (macOS)
- icon.ico (Windows)
- icon.png (System tray)

Or use an online tool like https://tauri.app/v1/guides/features/icons/
